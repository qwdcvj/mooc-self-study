# mooc-self-study

一个用于 MOOC 自学的浏览器用户脚本。它帮助你记录真实学习进度、恢复视频播放位置、在视频自然结束后进入下一节，并为文档页面提供自动翻页、朗读、笔记和摘要整理能力。

## 当前功能

- 视频自然播放结束后，自动查找并点击页面中的“下一节 / 下一课 / 继续学习 / Next”等按钮。
- 文档页停留 5 秒后，优先点击阅读器底部的页码右箭头或下一页缩略图。
- 文档没有下一页时，自动在“课件”的章节内容里进入下一项，例如从 `2.1` 进入 `2.2`，不会切到讨论区。
- 自动保存视频播放位置，刷新或返回课程页面后恢复到上次学习处。
- 本地记录视频完成时间、文档阅读进度和当前页面笔记。
- 文档页面可朗读正文或选中文字，适合边听边学。
- 自动提取文档标题和关键段落，一键复制为学习摘要。
- 支持时间戳笔记，方便回到视频中的关键位置复习。
- 点击“收起”后会保留右下角圆形“学”按钮，可随时重新打开助手。
- 屏蔽“提交、考试、测验、作业、签到、完成”等控件，避免脚本触碰考核或平台完成状态。

## 使用方式

1. 在浏览器安装 Tampermonkey 或 Violentmonkey 扩展。
2. 打开 `src/mooc-auto-next.user.js`。
3. 将脚本内容安装到用户脚本管理器。
4. 进入 MOOC 课程页面，正常播放视频或阅读文档。
5. 右下角会出现“MOOC 学习助手”面板，可控制自动下一节、恢复进度、朗读文本、复制摘要和记录笔记。

## 支持站点

脚本默认匹配以下站点：

- `icourse163.org`
- `*.icourse163.org`
- `imooc.com`
- `*.imooc.com`
- `xuetangx.com`
- `*.xuetangx.com`
- `chaoxing.com`
- `*.chaoxing.com`

如果你的课程页面不在这些域名下，可以在脚本头部新增 `@match` 规则。

## 常见问题

如果 Tampermonkey 菜单里显示“没有存取此页面的权限”，说明浏览器没有允许扩展在当前课程页面运行。请在浏览器扩展设置里把 Tampermonkey 的“网站访问权限”改为“在所有网站上”或允许访问当前 MOOC 域名，然后刷新课程页面。

如果脚本是在视频已经结束后才启用，0.2.1 起会检测已经处于结束状态的视频，并尝试进入下一节。

如果课程平台没有“下一节”按钮，而是把下一节显示成同一排里的相邻课程标签，0.2.2 起会在视频结束后尝试点击当前课程标签后面的第一个可用标签。

如果启用脚本后课程页面加载变慢，0.2.3 起已把页面变化监听和阅读进度扫描改为节流执行，减少对 MOOC 页面初始化的影响。

0.2.4 起，“完成后下一节”会在视频自然结束或文档阅读达到本地完成条件后触发。下一项查找会按顺序尝试：明确的下一节按钮、当前选中课程标签的相邻标签、同一课程导航容器里的下一个可点击课程项。

0.2.6 起，自动切换会排除讨论区、问答、评论、公告、分享、评价课程、老师提问等非课件入口，下一项只会优先在课件/章节内容中寻找。

0.2.7 起，文档页面会每秒检查一次；当前页停留 5 秒后先尝试翻到文档下一页，如果整篇文档没有下一页，则展开/读取课件章节菜单并进入同一课件列表里的下一项。

0.2.8 起，针对中国大学 MOOC 文档阅读器底部的 `2 / 4` 翻页栏和缩略图页码做了专门识别；课程下一项只接受比当前小节编号更靠后的课件项，减少误跳到非课件页面。

0.2.9 起，如果当前视口里没有看到文档翻页栏，脚本会先滑动到文档底部再找右箭头或下一页缩略图；最后一页没有下一页时，再回到顶部课件章节栏查找下一项。

0.3.0 起，自动滑动和自动跳转被硬限制在中国大学 MOOC 的 `课件 / learn/content` 页面内；“认证、成绩、证书”等入口会被排除。同时修复了课程名包含“安全评价”时被误屏蔽的问题。

0.3.1 起，点击文档“下一页”后会检查页码或页面内容是否真的变化；如果没有翻页会自动重试。确认已经到最后一页后，先回到顶部打开右侧小节菜单进入下一项，右侧没有下一项时再打开左侧章节菜单进入下一章。

0.3.2 起，脚本会识别 `4 / 4` 这类最后一页状态，不再反复滑到底部找不存在的下一页；右侧小节已经是最后一项时，会切换到左侧章节菜单并进入下一章。

0.3.3 起，脚本头部加入 GitHub Raw 的 `@downloadURL` 和 `@updateURL`，方便 Tampermonkey 从当前 GitHub 分支识别并更新到最新版。

0.3.4 起，只要识别到文档页码还没有到最后一页，脚本就不会寻找下一篇内容；找下一篇时会先点击右侧小节下拉按钮，右侧没有下一项时再点击左侧章节下拉按钮。

## 合规说明

本项目用于个人学习辅助和流程自动化。它不会快进视频、不会自动答题、不会提交测验、不会伪造平台学习完成状态。请遵守课程平台的用户协议和课程要求。
