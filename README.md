# mooc-self-study

登录 MOOC 慕课 App 后辅助进入课程、跟踪学习进度、观看课程视频，并生成学习总结。

## 当前功能

- 视频自然播放结束后，自动查找并点击页面中的“下一节 / 下一课 / 继续学习 / Next”等按钮。
- 支持常见 HTML5 `video` 播放器页面。
- 只做视频结束后的跳转，不快进视频，不自动答题。

## 使用方式

1. 在浏览器安装 Tampermonkey 或 Violentmonkey 扩展。
2. 打开 `src/mooc-auto-next.user.js`。
3. 将脚本内容安装到用户脚本管理器。
4. 进入 MOOC 课程视频页面，正常播放视频。
5. 当前视频自然结束后，脚本会尝试自动进入下一节视频。

## 支持站点

脚本默认匹配以下站点：

- `*.icourse163.org`
- `*.imooc.com`
- `*.xuetangx.com`
- `*.chaoxing.com`

如果你的课程页面不在这些域名下，可以在脚本头部新增 `@match` 规则。

## 注意事项

本项目用于个人学习辅助和流程自动化。请遵守课程平台的用户协议和课程要求，不用于绕过学习、考试、测验或评价流程。
